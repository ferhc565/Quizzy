QuizziBrain Android WebView

Proyecto de Android Studio que empaqueta la versión web de QuizziBrain dentro de una app nativa usando WebView.

Cómo usar:
1. Abre Android Studio.
2. Ve a "Open an Existing Project" y selecciona esta carpeta.
3. Deja que Android Studio descargue dependencias.
4. Conecta un teléfono o usa un emulador.
5. Ejecuta el módulo `app` (Run).

La app cargará el archivo local `file:///android_asset/www/index.html` con el quiz de QuizziBrain.
